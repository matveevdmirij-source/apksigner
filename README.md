APK Signer — Dmitry Matveev
Версия: 1.0
Дата: 2026

Это портативное приложение для:
- Подписи APK/AAB
- Перекомпиляции из debug в release
- Подготовки к публикации в Google Play и RuStore

КАК ИСПОЛЬЗОВАТЬ:

1. Запустите APK Signer/APK Signer.exe
2. Выберите APK или AAB файл
3. Укажите папку для сохранения
4. Нажмите "Генерировать ключи и подписи"



ВАЖНО:
- Если вы используете перекомпиляцию — убедитесь, что Java установлена
  (встроена в .exe, но apktool требует отдельного запуска java)
