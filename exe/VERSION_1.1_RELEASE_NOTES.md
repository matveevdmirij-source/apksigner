## 🔧 Технические подробности

### Алгоритм подписи
```
Алгоритм: RSA 2048-bit
Хеширование: SHA1 при подписи
Валидность: 10,000 дней (~27 лет)
Инструменты: keytool, jarsigner (встроены в Java)
```

### Процесс пересборки (Advanced)
```
Шаг 1: gradle clean          (очистка)
Шаг 2: gradle assembleRelease (сборка Release)
Шаг 3: Поиск Release APK     (автомат)
Шаг 4: jarsigner для подписи (подпись)
Шаг 5: Экспорт сертификата  (PEM)
Шаг 6: Генерация инфо маркетов (playmarket/rustore)

Время: зависит от размера проекта
Место: ~500 MB дополнительно (временные файлы)
```

### Требуемые инструменты
```
Java версии:
  ✓ Java 8+ (обязательно)  - уже установлена v11.0.17 ✓
  ✓ keytool (встроена в Java)
  ✓ jarsigner (встроена в Java)

Advanced версия ДОПОЛНИТЕЛЬНО:
  ✓ Gradle (нужно установить)
  ✓ Android SDK (если не встроен в проект)
  ✓ Примечание: Можно использовать gradlew из проекта

Python версии:
  ✓ Python 3.6+ (нужно установить)
  ✓ tkinter (встроена в Python)
  ✓ subprocess (встроена в Python)
```

## ✅ Проверка установки

### Для базовой версии (всегда работает)
```bash
java -version
# Должна показать: java version 11.0.17 или выше

javac -version  
# Должна показать: javac 11.0.17 или выше
```

### Для Advanced версии ДОПОЛНИТЕЛЬНО
```bash
gradle --version
# Должна показать версию gradle

# Или если gradle wrapper в проекте:
cd "путь_к_проекту"
gradlew.bat --version
```

### Для Python версии
```bash
python --version
# Должна показать: Python 3.6 или выше

python -m pip install -r requirements.txt
```
