# ⚡ БЫСТРО: За 1 минуту до результата!

## 🆕 Версия 1.2+ теперь поддерживает AAB!

**AAB (Android App Bundle)** - новый формат для Google Play с преимуществами:
- 40% меньше размер скачивания
- Автоматическая оптимизация для каждого устройства
- Лучше метрики

Если хочешь использовать AAB → [QUICK_AAB_START.md](QUICK_AAB_START.md)

---

## Вариант 1️⃣: У вас готов APK файл

```
1. Двойной клик на: run_java.bat
2. Выбрать формат: APK (или AAB если нужно)
3. Выбрать: файл (кнопка "Browse")
4. Нажать: "Генерировать ключи и подписи"
5. Ждать: 20-60 секунд
6. Результат: app-signed.apk готов!
```

✅ Результаты найдете в той же папке где был файл

---

## Вариант 2️⃣: У вас исходный код Android проекта

```
1. Двойной клик на: run_advanced.bat
2. Перейти: на вкладку "Пересборка Release"
3. Выбрать: папку Android проекта
4. Нажать: "Пересобрать и подписать Release"
5. Ждать: 5-30 минут (зависит от проекта)
6. Результат: готовый Release APK!

ВАЖНО: Пересборка создает APK (не AAB)
        Для AAB используйте Android Studio Bundle
```

✅ Результаты найдете в: проект/release_signed/

---

## Вариант 3️⃣: Что-то не работает?

```
1. Запустить: check_requirements.bat
2. Проверить: что Java установлена
3. Читать: TROUBLESHOOTING.md (найти вашу ошибку)
4. Выполнить: рекомендацию из документации
```

✅ Если всё еще не работает - смотрите INDEX.md
