## 📞 Глав проблем и решение

| Проблема | Решение | Рекомендация |
|----------|---------|-------------|
| Не запускается run_java.bat | Java не установлена? | Проверить: java -version |
| Ошибка "Cannot find keystore" | Неправильный путь | Выбрать существующий файл |
| Ошибка "gradle not found" | Gradle не установлена | Смотреть REBUILD_GUIDE.md |
| Ошибка подписи | Bad APK или keystore | Проверить TROUBLESHOOTING.md |
| Python не работает | Python 3 не установлен | Смотреть INSTALL.md |

→ Полное решение: читать [TROUBLESHOOTING.md](TROUBLESHOOTING.md)
